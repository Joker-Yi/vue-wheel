# 加载状态按钮

可以给按钮添加上图标，且图标位置left，right可供选择，默认left。

默认按钮
<template>
  <div>
    <vw-button :loading="loading" @click="onclick">默认按钮</vw-button>
  </div>
</template>
<script>
export default {
  data() {
    return {
      loading: false
    }
  },
  methods: {
    onclick() {
      this.loading = !this.loading
    }
  }
}
</script>
