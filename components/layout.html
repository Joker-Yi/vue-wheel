# 响应式布局

参照了 Bootstrap 的 响应式设计，预设了五个响应尺寸：phone（默认）,ipad,narrowPc,pc和widePc。

 
<template>
  <div>
    <vw-row>
        <vw-col span="24"
                :ipad="{span:6}"
                :pc="{span:8}"
        >
          <div class="temp"></div>
        </vw-col>
        <vw-col span="24"
                :ipad="{span:6}"
                :pc="{span:8}"
        >
          <div class="temp"></div>
        </vw-col>
