# ButtonGroup 按钮组

多个按钮形成的组合。

## 使用方法

::: demo 
```vue
<template>
  <div>
	<vw-button-group>
		<vw-button icon="zuo">上一页</vw-button>
		<vw-button >更多</vw-button>
		<vw-button icon="you" icon-position="right">下一页</vw-button>
	</vw-button-group>
  </div>
</template>
<script>
export default {
}
</script>
```
:::
