## 介绍

这是我在深入学习Vue2.0+中所写的一个UI框架，参考了element-ui、iview等popular UI框架，以助我进一步探知vue的魅力所在，欢迎学习和交流，如果对你有所帮助欢迎Star ！

## 开始使用
1. 安装

```
npm i yee-wheel-ui
```
